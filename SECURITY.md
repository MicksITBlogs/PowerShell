# Security

This repository contains Windows administration and endpoint management scripts. Some scripts can change BIOS settings, TPM state, BitLocker state, software installations, local admin membership, and ConfigMgr behavior.

## Reporting A Security Issue

Please do not open a public issue for a suspected security problem.

Use a private contact method from the maintainer GitHub profile, or a private repository security advisory if that feature is enabled for this repository.

## What To Include

1. Script name.
2. Clear description of the issue.
3. Impact.
4. Reproduction steps.
5. Environment details.
6. Any suggested mitigation.

## Scope Notes

1. Configuration mistakes in local lab usage are not always repository vulnerabilities.
2. Hard coded secrets, unsafe defaults, privilege escalation paths, or dangerous undocumented behavior should be reported privately.
